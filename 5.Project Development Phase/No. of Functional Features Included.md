*FinRelief AI — AI Powered Debt Relief & Financial Recovery Platform*

# Number of Functional Features Included
*Feature inventory for FinRelief AI*

---

## Feature Count Summary

The platform ships 9 core functional features across authentication, financial analysis, settlement prediction, and AI negotiation.

## Feature List

- 1. Borrower registration with hashed password storage.
- 2. JWT-based login and session management (120-minute expiry).
- 3. Loan creation, listing, and deletion.
- 4. Loan priority analysis and priority-sorted loan listing.
- 5. Financial profile creation/update.
- 6. Financial health metrics (EMI ratio, DTI ratio, surplus, stress level, health score).
- 7. Settlement prediction per loan (percentage, recommended amount, priority).
- 8. AI-generated negotiation strategy, settlement letter, and summary (Gemini + fallback).
- 9. Persisted AI interaction history, retrievable per borrower.

## Supporting (Non-Core) Features

- Know Your Rights static reference page.
- Responsive, dark-themed UI across all pages.
- Automatic API documentation via FastAPI's /docs endpoint.
