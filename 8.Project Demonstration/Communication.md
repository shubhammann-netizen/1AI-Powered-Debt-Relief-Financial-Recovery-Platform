*FinRelief AI — AI Powered Debt Relief & Financial Recovery Platform*

# Communication
*Communication plan for the project demonstration*

---

## Audience

Course instructor / evaluation panel, unfamiliar with the internal implementation but familiar with standard full-stack web concepts.

## Key Messages

- The problem: borrowers with multiple loans lack a clear, data-driven way to prioritize and negotiate debt settlements.
- The solution: FinRelief AI computes financial health, predicts settlements, and auto-generates negotiation letters using Gemini AI with a reliable fallback.
- The architecture: React frontend, FastAPI backend, SQLAlchemy/SQLite database, clearly separated by responsibility.

## Communication Channels

- Live screen-share walkthrough of the running application
- README.md for setup instructions
- This documentation set for supporting detail on design decisions

## Presenter Notes

Lead with the borrower's problem before showing any code, so the evaluation panel understands why each feature (priority sorting, settlement prediction, negotiation letters) exists before seeing how it is implemented.
