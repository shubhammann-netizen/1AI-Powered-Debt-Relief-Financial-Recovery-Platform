# Problem-Solution Fit
*Mapping borrower pain points to FinRelief AI capabilities*

---

## Pain Points Mapped to Solutions

| Pain Point | FinRelief AI Solution |
|---|---|
| No single view of total debt / urgency | Loan Priority Analysis + Priority-Based Loan Sorting (financial_engine.py) |
| Unclear financial standing | Financial Health Engine — EMI ratio, DTI ratio, surplus, health score |
| Doesn't know a realistic settlement amount | Settlement Prediction System (settlement_engine.py) |
| Can't write a persuasive negotiation letter | AI Negotiation Strategy Engine (Gemini + rule-based fallback) |
| No record of past negotiation attempts | AI_History table + History page |
| Unsure of borrower rights before contacting a lender | Know Your Rights reference page |

## Fit Validation

Each functional requirement traces directly to a specific backend module and frontend page, ensuring no pain point identified in the Empathy Map is left unaddressed by the shipped feature set.
