*FinRelief AI — AI Powered Debt Relief & Financial Recovery Platform*

# Technology Stack
*Technologies used across the FinRelief AI platform*

---

## Backend

- Python 3.11+ — core backend language
- FastAPI — REST API framework for authentication, loans, financial, settlement, and negotiation endpoints
- SQLAlchemy ORM — database modeling and querying
- SQLite — lightweight relational database (PostgreSQL-compatible via psycopg2-binary)
- python-jose — JWT creation and verification
- Werkzeug — password hashing
- python-dotenv — environment variable management
- google-generativeai — Google Gemini API client

## Frontend

- React.js — component-based UI library
- Vite — fast dev server and build tool
- Axios — HTTP client with JWT request/response interceptors
- React Router — client-side routing across dashboard pages
- Recharts — financial health and metrics visualizations

## Tooling & Collaboration

- Git & GitHub — version control and repository hosting
- Visual Studio Code — primary development environment
- Python venv — dependency isolation
